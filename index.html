BIENVENUE DANS LA MAISON DU BONHEUR

Afin de garder un bon souvenir de cette magnifique journée, Nous vous proposons une petite animation : Dans l'onglet "Vidéo", vous pourrez enregistrer un petit message vidéo via la webcam de l'oridnateur. D'un format d'nvirton 30 secondes, vous pourrez y exprimer tout ce qui vous amuse concernant les Guyons. Un montage final sera réalisé, et si vous le souhaitez il pourra vous etre envoyé
